# Elektroasas svetaine GitHub ikelimui

Kelk i GitHub visa sio aplanko turini, o ne sena projekta is `projektai` katalogo.

Ka kelti i repo:
- `index.html`
- `robots.txt`
- `sitemap.xml`
- `assets/`
- `images/`
- `elektrikas-kaune/`
- `elektros-darbai-kaune/`
- `elektros-instaliacija-kaune/`
- `avarinis-elektrikas-kaune/`
- `en/`
- `.nojekyll`

Ko nereikia kelti:
- seno darbinio aplanko is `projektai`
- Excel failu ar kitu nereikalingu dokumentu

Greiciausias budas:
1. Susikurk nauja GitHub repo, pvz. `elektroasas.lt`.
2. Atidaryk sita aplanka: `C:\Users\Nedas\OneDrive\Desktop\elektroasas-github-ready`
3. Pazymek visus failus ir aplankus viduje.
4. Ikelk juos i GitHub repo pagrinda (root), ne i papildoma poaplanki.
5. Jei naudoji GitHub Pages, publishink is branch `main` root katalogo.

Svarbu:
- Jei ikelsi visa aplanka kaip viena poaplanki repo viduje, nuorodos gali neveikti.
- Ikelti reikia butent sio aplanko VIDU, ne pati aplanka kaip subfolderi.
